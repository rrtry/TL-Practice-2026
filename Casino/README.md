# Casino

Задача - реализовать игру, в которой пользователь вводит ставку в консоль, далее выполняется механизм расчета выигрыша, начальный баланс вводится пользователем при старте
Механизм расчета:
Генерируем случайное на отрезке [1, 20]
Выигрышем будем считать значения 18, 19, 20
В случае проигрыша - забираем ставку
В случае выигрыша - считаем сумму выигрыша по формуле

`{bet} * (1 + ({multiplicator} * {random_num} % 17))`
